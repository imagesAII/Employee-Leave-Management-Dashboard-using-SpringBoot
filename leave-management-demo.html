📝 Leave Apply Flow
1️⃣
LeaveController.applyLeave()
Receives validated DTO, extracts email from JWT context
2️⃣
Date Validation
Start ≥ today, end ≥ start — throws LeaveException if invalid
3️⃣
Overlap Check
Queries DB for any existing APPROVED leave in the same range
4️⃣
Balance Check
Working days calculated (Mon–Fri), compared against LeaveBalance
5️⃣
Saved as PENDING
Stored in DB, awaits manager review via reviewLeave()
